# Assets

This directory contains static assets like images, fonts, and other media files.

## Structure

- `images/` - Image files (PNG, JPG, SVG)
- `fonts/` - Custom font files (TTF, OTF)

## Usage

### Images

```typescript
import logo from '../assets/images/logo.png';
```

### Fonts

1. Add font files to this directory
2. Link fonts in your native projects (iOS/Android)
3. Use fonts in your styles

## Best Practices

- Optimize images for mobile
- Use vector graphics (SVG) when possible
- Organize assets by category
- Use descriptive file names
